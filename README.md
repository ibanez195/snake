# Snake
This is an implementation of the classic game 'snake' using ncurses for the
linux terminal

## Compilation
Enter repository directory and run

```
make
```

## Instructions
Run program with
```
./snake
```
### Controls
Control the snake with the arrow keys
p - pause game
